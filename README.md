hapi-h2
=================================================

Demo hapi17 server setup with http2 protocol

## Run in local
#### install node_modules
```
npm i
```

#### build bundle.js
```
npm run build
```

#### run http1 in http://localhost:3000
```
node h1-server.js
```

#### run http1 in https://localhost:3001
```
node h2-server.js
```
